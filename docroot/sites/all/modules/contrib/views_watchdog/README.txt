
Module
======
Views Watchdog


Description
===========
This module extends the Views module and allows to create customized lists
(pages, blocks, feeds) of watchdog entries.


Requirements
============
- Drupal 7.x
- Views 7.x-3.x


Installation
============
1. Move this folder into your modules directory.
2. Enable it from Administration >> Modules >> Views.


Configuration & Usage
=====================
Create customized lists at Administration >> Structure >> Views.

For more information, see the online handbook entry at
http://drupal.org/node/1225540
